Яндекс Геокодер и Поиск по организациям

Продолжаем знакомиться с сервисом Яндекс.Карты. Сегодня поговорим про "Геокодер" и "Поиск по организациям". Их работа достаточно схожа. В отличие от Яндекс Static.Maps API эти сервисы будут возвращать JSON. Напомню, что Геокодер нужен для сопоставления координат объекта с топонимом (название страны, города, горы, озера и т.д.). Поиск по организациям же делает аналогичную работу, но уже не с топонимами, а с организациями. Так же эти сервисы отличаются от Яндекс.Maps Static своей "закрытостью". К счастью мы можем получить бесплатный ключ для каждого из сервисов.

Для этого вам необходимо либо войти в свою Яндекс почту либо её создать. После чего вам нужно перейти в кабинет Яндекс разработчика и нажать на кнопку "Подключить API". Выберите "JavaScript API и HTTP Геокодер" и нажмите продолжить. Заполните все необходимые поля. Думаю, что если вы сильно озабочены своей безопасностью, то не очень критично вводить свои настоящие данные, так как вы будете использовать API только для обучение. Хотя с другой стороны судебных разборок с Яндексом вам, думаю, тоже не надо, поэтому думайте сами. Так же выбираете:

  • использование API в открытой системе,
  • в бесплатном варианте, Буду отображать данные на карте,
  • в качестве ссылки на сайт можете указать ссылку на мой курс (или просто www.example.com),
  • в кратком описании сервиса укажите просто "обучение",
  • ставим галку, что принимаем условия лицензии
  • вводим капчу.

Всё, после этого вы получили ключ для геокодера на 25000 запросов в сутки.

Аналогично можно получить ключ и для поиска по организациям. Форма там даже проще.

Получите  оба ключа. 

Теперь ознакомьтесь с документацией для этих сервисов (Геокодер, Поиск по организациям). Начать рекомендую с геокодера. Сначала протестируйте примеры из документации просто в командую строку. Не забудьте указать формат данных JSON (format=json в get запросе).  При этом вы будете получать страницу с абсолютно нечитаемым JSON на весь экран. Для удобство воспользуйтесь каким-нибудь сайтом, или редактором. В качестве удобства могу посоветовать, например

  • такой сайт: http://jsonviewer.stack.hu/
  • программку postman для отправки http запросов
  •  расширение для chrome JSON-Viewer или аналогичное ему

Ваша основная задача - разобраться в структуре ответа, который к вам приходит.
