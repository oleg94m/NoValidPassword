Поворот на 90

Вводятся 2 натуральных числа: n и m. Затем вводится двумерный массив размером n*m. Поверните его на 90 градусов по часовой стрелке и выведите на экран (через пробел).


Sample Input:
3
4
1 2 3 8
4 6 7 8
-5 6 3 4

Sample Output:
-5 4 1 
6 6 2 
3 7 3 
4 8 8



Memory limit: 256 MB
Time limit: 8 seconds
