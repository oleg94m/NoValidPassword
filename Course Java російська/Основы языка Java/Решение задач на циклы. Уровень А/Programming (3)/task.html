Доллары

Напишите программу, которая выводила бы на экран числа (например число а) следующим образом:

a a.$

Входные данные
Вводятся два натуральных числа a и b (a <= b).

Выходные данные
Выведите ответ на задачу.


Sample Input:
10
25

11 11.$

13 13.$
14 14.$
15 15.$
16 16.$
17 17.$
18 18.$
19 19.$
20 20.$
21 21.$
22 22.$

25 25.$



Memory limit: 256 MB
