Сумма чисел

Напишите программу, которая находит сумму натуральных чисел от 1 до n, где n вводится с клавиатуры.

Входные данные
Задано натуральное число n 

Выходные данные
Выведите единственное число - ответ на задачу.


Sample Input:
10

Sample Output:
55


Sample Input:
5

Sample Output:
15



Memory limit: 256 MB
Time limit: 8 seconds
