Редакторы кода

Редакторы кода обычно обладают гораздо более скромным функционалом и представляют из себя блокнот на стероидах. Они умеют подсвечивать код и делать небольшие исправления но, как правило, не умеют запускать программу, отлаживать её и делать более сложные операции. Зато они умеют редактировать и воспринимать код абсолютно на любом языка. То есть по отношению к гигантским средам разработки они являются более всеядными. Кстати, некоторые из них всё-таки умеют запускать ваш код, если вы их правильно настроите. Самые популярные из них:

  1. Sublime Text 3

  2. Atom

  3. Visual Studio Code

  4. Brackets

Среды Разработки

Как правило, среды разработки представляют из себя огромный набор инструментов для разработчика. От подсветки кода и автоматического поиска ошибок до автоматического генерирования кода, детальной отладки и написания тестов. Чаще всего они создаются для ограниченного числа языков. Тут представлены самые популярные среды разработки для языка Java.

1. NetBeans  -  инструкция по установке и первая программа

 

2. Eclipse -  инструкция по установке и первая программа

 

3. IntelliJ IDEA -  инструкция по установке и первая программа
