Комментарии 

Комментарии - неотъемлемая часть жизни любого программиста. Они нужны для того, что бы пояснять сложные (а может, и не очень) части программы. По сути это просто заметки разработчика, оставленные самому себе или кому-либо ещё. Также можно комментировать и сам код, если вы не хотите, чтобы он выполнялся.

Java поддерживает и однострочные, и многострочные комментарии. Все символы, которые находятся внутри комментариев, игнорируются Java компилятором.

Однострочные комментарии 

Начинается с двух косых черт (слэшей) и продолжается до окончания строки.

Многострочные комментарии

Java также поддерживает комментарии, которые занимают несколько строк. Такой комментарий начинается с символа косой черты, за которым следует символ звездочки, и заканчивается символом звездочки, за которым следует символ косой черты.

 

class Test_comment{
    /*
       привет, я многострочный комментарий
       я могу занимать столько строк, сколько захочу
    */
    public static void main(String[] args){
        //а я однострочный комментарий и занимаю только одну строчку
        System.out.println("Комментарии"); // эта команда выводит строку
        //следующую строку я не хочу выполнять, но и удалять тоже не хочу
        //System.out.print("Бяка");
    }
}
