Задача 5

Пользователь вводит натуральные числа по одному на каждой строке. Когда ему надоедает, то он вводит 0. Найти количество введенных чисел (0 не считается)


Sample Input:
8
5
1
4
9
9
0

Sample Output:
6


Sample Input:
10
2
23
0

Sample Output:
3



Memory limit: 256 MB
Time limit: 8 seconds
