Учитесь искать инфу сами

На этом уроке новой информации почти не будет. Из его названия понятно, что строку (String) можно представить как массив символов (char) и работать с ним. Сегодня мы с вами попрактикуемся в этом. Функцию toCharArray вы уже увидели на прошлом уроке. Мы уже порядком поработали с массивами, состоящими из чисел. Теперь попрактикуемся на символах. Могу посоветовать вам перед началом решения задач немного подготовиться и найти в интернете, что делают следующие методы:

  • Character.toLowerCase 
  • Character.toUpperCase 
  • Character.isDigit 
  • Character.isLetter 
  • Character.valueOf 

Да и вообще неплохо бы поближе познакомиться с классом Character.
