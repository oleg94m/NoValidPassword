Длина массива

Если вдруг оказалось, что у вас есть массив, но вы не знаете его длину, то можно её найти с помощью команды length.

int[] a = {5, 3, -4, 8, 5, -1, -2, -9, 0, 5, 4, 3, -4, 5, 3, 2, 5, -5, 4};
int n = a.length; // в n лежит число 19
