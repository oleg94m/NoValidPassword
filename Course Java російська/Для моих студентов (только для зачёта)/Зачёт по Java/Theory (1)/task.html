5. Википедия

Напишите программу с использование графического интерфейса и API википедии. в программе должны быть минимум три виджета: 

  • текстовое поле для ввода запроса
  • кнопка "поиск"
  • большое текстовое поле для вывода ответа (JTextArea)

Человек вводит запрос, и нажимает на кнопку поиск. После этого ваша программа должна найти информацию о запросе на сайте википедия.

Задание проверяется преподавателем.
