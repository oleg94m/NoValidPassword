3. Строковый Камаз

Вводится целое число n, после чего вводится n строк. Ваша задача определить, скольких строках есть упоминание о камазе в любом регистре. Обратите внимание, что если в одно строчке камаз упоминается несколько раз, то это всё-равно одна строка.


Sample Input:
3
О Камаз ты мой, камаз
Странно было думать, что я смогу жить без тебя...
Мой КаМаЗИК!!!

Sample Output:
2



Memory limit: 256 MB
Time limit: 8 seconds
