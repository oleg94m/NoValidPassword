Интерфейс ActionListener

В java может происходить множество различных событий. для них существует удобный обработчик - ActionListener, который может вызвать ту или иную функцию тогда, когда вам нужно. Его важная отличительная черта - универсальность, так как он подходит для любого типа событий.
