SQL запросы из Java

Для работы с базами данных в языке Java существует специальная технология JDBC. Это универсальный инструмент для работы с SQL - запросами и их выполнения в той или иной БД. С некоторыми БД можно работать "из коробки", но для большинства нужно подключить особый драйвер, который предоставляют разработчики той или иной СУБД. Для работы с базами данных SQLite нам понадобится такой драйвер. Предварительно его нужно скачать и распаковать. После чего библиотеку (наш драйвер) нужно подключить к проекту. Вот пример, как это можно сделать в Intellij Idea.
