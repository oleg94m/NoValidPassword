ВНИМАНИЕ! Важная информация!

Дальнейшее проведение курса предполагает гораздо более плотное взаимодействие преподавателя и учеников. Всё ООП педагог объясняет самостоятельно на примерах, разбираю каждую строчку. Поэтому дальнейшие слайды будут представлять из себя наборы ссылок на сторонние источники, чтобы прочие обучающиеся могли получить весь объём информации. 

Так же, к сожалению, Степик как-то плохо проверяет ответы, в которых много строк с кириллическими символами. Поэтому теперь задачи записываются исключительно для самоконтроля.
