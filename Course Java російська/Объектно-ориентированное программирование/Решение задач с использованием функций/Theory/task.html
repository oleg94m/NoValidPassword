Особенности работы с изменяемыми типами данных

Пример:

class Programm {

    public static void main(String[] args) {
        int[] mas = {1,2,3,4};
        f(mas);
        System.out.println(mas[0]);
    }

    static void f(int[] a){
        a[0] = 10;
    }
}

Попробуйте запустить код, написанный выше. Обратите внимание, что выведется на экран. Такая работа связана исключительно с ссылочными типами данных. Вы передаёте в функцию не само значение, а только ссылку на него. Поэтому, после изменения переменной a внутри функции, меняется и значение переменной mas внутри функции  main. То есть функция ничего не возвращает, но может изменить значение ссылочного типа. Однако, со строкой так не получится, потому что она является неизменяемым типом данных. А массив и прочие коллекции - изменяемый. Изменяемыми называют такие типы данных, у которых возможно изменить содержанию внутри переменных, не занося туда новое значение. Обратите внимание, что мы меняем не весь массив, а только его часть. Когда мы добавляем элемент в список мы меняем содержимое списка, но при этом не заносим в саму переменную нового значения.
