Write a function which computes all 3 means

Did you know there are 3 different ways of computing means? Typically we use mean to specify the arithmetic mean, but it could also be the geometric mean, or the harmonic mean. The algorithm to compute the arithmetic mean of n values is:

arithmMean ← (value1 + value2 + ... + valuen) / n

The algorithm to compute the geometric mean of n values is:

geoMean ← pow(value1 * value2 * ... * valuen, 1.0/n)

The algorithm to compute the harmonic mean of n values is:

harMean ← n / (1.0/value1 + 1.0/value2 + ... + 1.0/valuen)

For your program, write a function, named computeMean, that has 7 parameters of type double. The first five parameters are the values whose means we will compute and should be passed-by-value. The last two parameters should be passed-by-reference and will be used to return the geometric mean and harmonic mean of the first five parameters. Your function should also return a double equal to the arithmetic mean of the first five parameters.

Write your main() function so that uses a loop that repeats 3 times. Each time through your loop, your program should read in all 5 values, call computeMean and then print out the 3 different means with a space after each. Do NOT print out a newline (or endl) at the end of your cout)



Memory limit: 256 MB
Time limit: 5 seconds
