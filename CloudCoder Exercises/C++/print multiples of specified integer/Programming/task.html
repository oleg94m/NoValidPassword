print multiples of specified integer

The program will receive two integer input values. The first is an arbitrary integer value, N. The second is a count. Based on these values, the program should print the multiples of the first integer (N) from 1 to count.

For example, if the input values are 3 8, then the output of the program should be 

1x3=3 2x3=6 3x3=9 4x3=12 5x3=15 6x3=18 7x3=21 8x3=24

Another example: if the input values are 7 5, then the output of the program should be 

1x7=7 2x7=14 3x7=21 4x7=28 5x7=35

Hints:

  • Declare variables for the two input values, N and count
  • Use a for loop to count from 1 to count
  • Inside the body of the loop, declare a variable for the multiple that will be computed
  • Think about how to compute each multiple based on the loop counter variable
  • Use printf to print each multiple, making sure that there is at least one space after each
  • Make sure the output is a single line


Memory limit: 256 MB
Time limit: 5 seconds
