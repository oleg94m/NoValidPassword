3...2...1...blast off!

The program should read a single integer input value, which you can assume will be positive. It should then print a message that counts down from that integer to 1, printing a message similar to 

3...2...1...blast off!

but starting from the input value.

Example: if the input is 5, the output should be

5...4...3...2...1...blast off!

Example: if the input is 2, the output should be

2...1...blast off!

Hints:

  • Make sure the output is on a single line
  • Declare a variable to store the input value
  • Use scanf to read the input value
  • Use a for loop to count down from n to 1, where n is the input value
  • Print three dots after each integer value
  • Print "blast off!" at the end of the line


Memory limit: 256 MB
Time limit: 5 seconds
