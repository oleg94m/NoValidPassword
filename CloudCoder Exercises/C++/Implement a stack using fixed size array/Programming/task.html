Implement a stack using fixed size array

Fixed array as Stack

The goal: Implement the functions push(), pop() and peek() for a stack that is implemented using a fixed integer array. Read the comments below for more details.

// Using global variable to define the stack, for convenience
// But generally, this is *not* a recommended coding style
//
int stack[10]; // this stack can hold a maximum of how many items?
int top = -1;  // if top == -1, it indicates that the stack is empty

push()

push() must accept an integer val as an integer parameter, and push it on top of the stack. If there should be no space on the the stack, then the function should return -1. Also, print Stack overflow! to indicate overflow condition.

int push(int val) { /* your code goes here */ }

pop()

pop() should fetch the integer element on top of the stack and remove it off the stack as well. If there is nothing in the stack, then the function should return -1. Also, print Stack underflow! to indicate this condition.

int pop() {  /* your code goes here */ }

peek()

peek() should only fetch the integer element on top of the stack, but not remove it from the stack. If there is nothing in the stack, then the function should return -1. Also, print Stack empty! to indicate this condition.

int peek() {  /* your code goes here */ }

Functions already defined

Both the below functions behave true to their names. Both of them take a count value so that many number of items can be read from the stdin, and printed to the stdout respectively.

void populate_the_stack(int);   // reads values from stdin
void display_the_stack(int);    // prints all stack elements and pops them
