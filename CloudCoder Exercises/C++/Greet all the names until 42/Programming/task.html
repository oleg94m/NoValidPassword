Greet all the names until 42

Indefinite Loops

Using a while loop greet as many names as that are available on the stdin. Stop when you read the string '42' as a name. Precede them with the sequence count.

Example

Dhoni Sachin -> 1 Hello Dhoni!2 Hello Sachin!

"42" -> "" (and exit)



Memory limit: 256 MB
Time limit: 5 seconds
