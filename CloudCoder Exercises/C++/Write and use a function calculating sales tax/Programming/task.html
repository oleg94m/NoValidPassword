Write and use a function calculating sales tax

Write a function whose prototype is:
double calcTotalWithTax(int countyId);
. This function read in the cost of an item (this will be a decimal). The function should then compute the tax that will be charged owed. This tax depends on the value of countyId: if countyId is less than 3, then the tax is equal to the cost of the item times 0.04 (e.g., only the 4% NY state sales tax). If countyId is between 3 and 10, then the tax is equal to the cost of the item time a 0.06 (e.g., the state sales tax AND a 2% county tax). Otherwise, e.g., countyId is greater than 10, the tax is equal to the cost of the item times 0.0875 (e.g., the state sales tax AND the maximum 4.75% county tax). Finally, calcTotalWithTax should return the sum of the item's cost and the tax you computed.

Your program will also need a main() which reads in the county ID (this will be an integer), calls calcTotalWithTax and prints out the value it returns formatted to show exactly 2 places to the right of the decimal point.



Memory limit: 256 MB
Time limit: 5 seconds
