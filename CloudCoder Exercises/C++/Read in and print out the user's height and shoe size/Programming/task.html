Read in and print out the user's height and shoe size

Write a program which begins by reading a user's height (this will be in meters and so may not be an integer). The program should then read in the user's shoe size (which, again, may not be an integer). Finally, your program should print out:

You are height meters tall and wear a size shoeSize shoe.

where height and shoeSize are the numbers the user typed in. Because we must be accurate, the height must be printed out showing 1 place to the right of the decimal point.



Memory limit: 256 MB
Time limit: 5 seconds
