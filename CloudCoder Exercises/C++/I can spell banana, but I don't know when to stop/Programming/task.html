I can spell banana, but I don't know when to stop

The program should read in a single integer input value, which you can assume will be non-negative. It should then print "ba" followed by n repetitions of "na", where n is the input value. Note that all of the output should appear on a single line with no spaces.

Example: if the input value is 2, the output should be 

banana

Example: if the input value is 6, the output should be 

banananananana

Hints:

  • Declare a variable in which to store the input
  • Use scanf to read the input value
  • Use a for loop to count from 1 to n (where n is the input value)


Memory limit: 256 MB
Time limit: 5 seconds
