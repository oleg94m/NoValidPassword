Compute a color the user would tolerate.

First declare a struct which has 3 fields of type int named red, green, and blue. Your program should declare 3 variables of that struct type named like, dislike, and meh. Your program should read the red, green, and blue values of their favorite color into the appropriate fields of like. The program should then read the red, green, and blue values of the ugliest color into the appropriate fields of dislike. Have your program set each field in meh to be the average of the values of that field in like and dislike. Finally, print out a message stating saying "You'll be neutral about" and print out (on the same line and separated by a space) meh's red, green, and blue fields.



Memory limit: 256 MB
Time limit: 5 seconds
