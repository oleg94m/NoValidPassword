Better way to handle data to average

Did you remember the 3 different ways of computing means? For those of you who have blocked all memories of last week from your mind, they are the arithmetic mean, geometric mean, and harmonic mean. The algorithm to compute the arithmetic mean of n values is:

arithmMean ← (value1 + value2 + ... + valuen) / n

The algorithm to compute the geometric mean of n values is:

geoMean ← pow(value1 * value2 * ... * valuen, 1.0/n)

The algorithm to compute the harmonic mean of n values is:

harMean ← n / (1.0/value1 + 1.0/value2 + ... + 1.0/valuen)

For your program, write a function, named computeMean, that has 4 parameters: an array of doubles, an int specifying the length of the array, and TWO pass-by-reference parameters of type double to return the geometric mean and harmonic mean of the array entries. Your function will also return a double equal to the arithmetic mean of the array's entries. You then need to write your main() function so that it creates two double arrays: one with a length of 2 and one with a length of 3. Read in the first 2 values into the first array and use this in a call to computeMean(). Print out the arithmentic mean, geometric mean, and harmonic means it computes (separate them by spaces). Then read in the 3 numbers into your other array, use it in a call to computeMean, and print out the arithmetic mean, geometric mean, and harmonic mean computed then (again separated by spaces). ALL 6 NUMBERS SHOULD APPEAR ON THE SAME LINE.
