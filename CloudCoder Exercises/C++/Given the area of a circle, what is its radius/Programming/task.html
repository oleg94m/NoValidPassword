Given the area of a circle, what is its radius?

The program should read a single double value, which is the area of a circle. It should then print the radius of the circle with that area.

The area (a) of a circle is given by the equation

a = π r2

where r is the radius. Rearranging this equation, we get

r = √ a / π

For example, if the input area is 4.0, then the output of the program should be 

Radius is 1.128379

Hints:

  • Declare variables with type double to store the area and radius
  • Use the sqrt function to compute a square root
  • The M_PI constant is a close approximation of π
  • Use the %lf format specifier to read a double value using scanf and to output a value using printf


Memory limit: 256 MB
Time limit: 5 seconds
