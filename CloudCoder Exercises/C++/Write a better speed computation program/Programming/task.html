Write a better speed computation program

Write a program which reads in a distance (it will be an integer). If the distance is less than 0, your program should print out (but do not move onto the next line): "This isn't physics, buddy. Distances must be positive." and set the distance equal to -1 times the variable's current value. No matter what, your program should then compute the user's speed if the drive took 3 hours. (For those who do not remember, the speed would equal the current value of the distance divided by 3). You should then print out:

Even Sherman didn't burn through Georgia like that! You drove speed mph for distance miles.

where speed is the speed you calculated and distance is the distance the user typed in. Because we want to be precise, the speed should be printed out showing 2 places to the right of the decimal point.



Memory limit: 256 MB
Time limit: 5 seconds
