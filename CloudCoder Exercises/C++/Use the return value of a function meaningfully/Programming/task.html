Use the return value of a function meaningfully

Functions

  • Read in a greeting. Read in a name to greet. Both on the standard in.
  • Print the greeting using both the strings (greeting and name).
  • Then write a function to print the two strings, one letter at a time, with a space in between.

Use strlen() (from the string library) to determine the length of the strings.

Make the function return the number of characters it printed. In the main function, print the value returned from the function.

Example

  • "Hello" and "World!" -> "Hello World! H e l l o W o r l d ! ---11"
  • "Hello" and "Sachin!" -> "Hello Dhoni! H e l l o S a c h i n ! ---12"
