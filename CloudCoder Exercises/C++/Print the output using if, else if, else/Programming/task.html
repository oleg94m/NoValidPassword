Print the output using if, else if, else

Given a variable, write a C++ program to print "Number one is Bigger" if the number one is bigger, else if number two is bigger "Number two is Bigger" , otherwise print "Number three is Bigger".

10, 20, 30 -> Number three is Bigger

20, 10, 15 -> Number one is Bigger

10, 80, 20 -> Number two is Bigger 



Memory limit: 256 MB
Time limit: 5 seconds
