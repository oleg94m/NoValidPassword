Summate only a subsection of the array

Return the sum of the numbers in the array, except ignore sections of numbers starting with a 4 and extending to the next 5 (every 4 will be followed by at least one 5). Return -1 for no numbers.
sum_45({1, 2, 2}, 3) → 5 sum_45({1, 2, 2, 4, 99, 99, 5}, 7) → 5 sum_45({1, 1, 4, 5, 2}, 5) → 4
 
Take this one slow and methodically. Put down an 'algorithm' on paper before you start writing code. Consider writing some helper functions. Bonus points for code that explains itself. Use of a 'do-while' loop wil help the cause.


Memory limit: 256 MB
Time limit: 5 seconds
