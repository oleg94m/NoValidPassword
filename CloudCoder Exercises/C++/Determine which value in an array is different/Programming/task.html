Determine which value in an array is different

Complete the program so that it finds the index of a unique value within an array containing all but one identical values.

For example, for 5, 5, 5, 5, 3, 5, 5, 5, 5 the output would be 

3 is the different value at index 4

Another example, for 1, 3, 3, 3, 3, 3, 3 the output would be 

1 is the different value at index 0

Another example, for 6, 6, 6, 6, 5 the output would be 

5 is the different value at index 4

Comments describe how to proceed. You should replace each "TODO" comment with appropriate code.

Hints:

  1. The array is named num[ ] and the number of elements it contains is stored in a variable size.
  2. Store the different value in the variable val and the index the value occurs at in the variable idx.
  3. Use a for loop to proceed through the array comparing the previous, current, and next elements to determine if the current element is different than the other two, i.e. is the different one.
  4. Consider how to handle the first or last elements being the different ones.

Note that the test inputs include the number of values the array will contain as the first input.



Memory limit: 256 MB
Time limit: 5 seconds
