use a function to generate a diamond

Note: This exercise is fairly challenging!

Add a prototype and definition for the choose function. The prototype should look like this: 

char choose(int x, int y, int size);

The definition of the choose function should cause the code in the main function to print a "diamond" (generated from 'X' and '-' characters) whose size (width and height) is determined by an integer value read as input. For example, if the input is 5, then the program's output should be: 

--X--
-XXX-
XXXXX
-XXX-
--X--

Another example: if the input is 6, then the output should be 

--XX--
-XXXX-
XXXXXX
XXXXXX
-XXXX-
--XX--

Important: do not modify the main function in any way.

Hints:

  • Click the "Show all" button in the Output column to see your program's output for a particular test case.
  • Try getting the program to work for odd sizes first, then modify it to work for even sizes.


Memory limit: 256 MB
Time limit: 5 seconds
