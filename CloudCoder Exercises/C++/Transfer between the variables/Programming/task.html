Transfer between Variables

  • Step 0 Define the necessary variables
  1. Read a number (variable_1) on the standard in
  2. Read another number (variable_2) on the stdin
  3. Read a third number (variable_3) on the stdin
  4. Variable_4 is to be assigned a value of 1000 always
  5. Print the values of the 4 variables (ALREADY PROVIDED, DO NOT EDIT)
  6. Transfer the value between the variables, so that
    • the value of variable_1-> variable_2
    • the value of variable_2 -> variable_3
    • the value of variable_3 -> variable_4
  7. Variable_1 to be assigned a value of 100
  8. Print the values of the 4 variables (ALREADY PROVIDED, DO NOT EDIT)

Example

If 100, 200 and 300 are inputs, then the expected output is

  • 100 200 300 1000 (expected output as per Step 5)
  • 100 100 200 300 (expected output as per Step 8)


Memory limit: 256 MB
Time limit: 5 seconds
