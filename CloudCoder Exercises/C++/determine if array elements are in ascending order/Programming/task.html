determine if array elements are in ascending order

Write a function called IsAscending. It should take two parameters:

  • an array of double elements
  • an integer, specifying how many elements are in the array

The function should return an int value as follows:

  • if the array is in asending order, meaning that each element is greater than the element preceeding it, the function should return 1
  • otherwise, the function should return 0

Hint: the function should scan through pairs of values.

As a special case, if the array has no elements, the function should return 1.



Memory limit: 256 MB
Time limit: 5 seconds
