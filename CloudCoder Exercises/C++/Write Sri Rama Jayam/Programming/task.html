Write Sri Rama Jayam

Definite Loops

  • Print "Sri Rama Jayam" exactly as many times as the value provided on the stdin.
  • Precede the mantra recital with its corresponding count.
  • For this exercise, you may either start the count starting from 0, or 1.

Example:

2 -> "1 Sri Rama Jayam 2 Sri Rama Jayam "

1 -> "1 Sri Rama Jayam "



Memory limit: 256 MB
Time limit: 5 seconds
