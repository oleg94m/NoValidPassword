Compute the average of two numbers

The program should read 2 double values, and then print the average. For example, if the input values are 5.0 and 10.0, then the output of the program should be 

The average is 7.500000

Hints:

  • The average of two numbers is their sum divided by 2
  • Use scanf to read the user values
  • Use printf to print the output
  • The %lf conversion can be used with scanf/printf to read/output a double value


Memory limit: 256 MB
Time limit: 5 seconds
