Count the number of odd integer values in an array

Write a function called countOdd that takes two inputs:

  • an array of integer elements
  • an integer, specifying how many elements are in the array

The function should count and return the number of odd integer values in the array. If there are no odd integer values in the array, the return value should be 0.

Hint: remember that you can use the modulus operator, %, to determine if a value is even or odd.



Memory limit: 256 MB
Time limit: 5 seconds
