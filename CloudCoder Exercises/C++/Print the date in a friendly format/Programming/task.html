Print the date in a friendly format

Complete the program so that it reads three integer values using scanf and then prints the following message:

The date is: M/D/Y

where M is an integer representation for the month, D is the day of the month and Y is the year.

The program should read the three integer values in the following order:

  • first, read an integer that specifies the month
  • second, read an integer that specifies the day of the month
  • third, read an integer that specifies the year

For example, if the program reads the three integers 1, 24, and 2013, then the output should be: 

The date is: 1/24/2013

Comments describe how to proceed. You should replace each "TODO" comment with a single line of code.

Hints:

  1. Declare an int variable in which to store the integer value for the month
  2. Declare a second int variable in which to store the day of the month
  3. Declare a third int variable in which to store the year
  4. Use three separate scanf statements to read each of the three integer values (see previous CloudCoder problems if you need additional help with scanf)
  5. Recall that a printf statement can include multiple %i placeholders to print multiple integers using a single printf statement


Memory limit: 256 MB
Time limit: 5 seconds
