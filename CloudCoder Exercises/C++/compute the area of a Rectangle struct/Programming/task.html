compute the area of a Rectangle struct

The Rectangle struct represents a rectangle. It has four fields, x1, y1, x2, and y2, which represent the x and y coordinates of the two corners of the rectangle.

In the main main function, an instance of the Rectangle struct called r is declared, and its field values are read from input values. Your task is to use the field values in r to compute the area of the rectangle.

For example, if the rectangle has the field values x1=2.0, y1=1.0, x2=6.0, and y2=4.0, then the area is 12.0, and the output of the program should be 

Area is 12.000000

The printf statement to print the output is already included. You will just need to make an assignment to the area variable.
