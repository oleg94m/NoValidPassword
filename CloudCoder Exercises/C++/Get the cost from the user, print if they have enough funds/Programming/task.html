Get the cost from the user, print if they have enough funds.

The user will start out with $100.00 in their bank. The program should read the cost of item they are buying. If the cost of the item is more than the funds in the bank Then the program should print insufficient funds. Otherwise The program should print the balance in their bank. For example, if the input value is 110.00, then the output of the program should be 

Insufficient Funds

if the input value is 65.00, then the output of the program should be 

Your bank balance is $35.00

Hints:

  • You will need to use an if statement to check if the cost of the item is greater than the amount in the bank


Memory limit: 256 MB
Time limit: 5 seconds
