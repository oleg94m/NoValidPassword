Read data into an array and then print it out

Your program only needs a main() function. Your main() function should declare an array of 8 doubles. Write a loop which repeats once per entry in the array. Each iteration of this loop should read a decimal number into the array (so the number typed in during the 1st iteration should be stored at index 0, the number typed into during the 2nd iteration would be stored at index 1, and so on). After the loop is finished, print out all 8 values that are stored in the array (separated by spaces) and then (after another space) use the built-in sizeof function to print out the total space required for the array.



Memory limit: 256 MB
Time limit: 5 seconds
