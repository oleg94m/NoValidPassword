Return the index of the last substring in the string or -1

indexOf

Takes three arguments. The first one is the target and the other one is the string to find. If find is available within target, then return the index at which it occurs.

  • If the third argument first is 1, then return the index of the first occurrence. Otherwise, return the index at which the last occurrence was matched.

If find is not available, then return -1.

Example

  • "abcde", "ab" -> 0
  • "abcdexy", "xy" -> 5
  • "abcde", "12" -> -1

Functional
