Generate the prime factors of the given number

Prime Factors

Inspired by one of Uncle Bob's TDD Kata

Write a program that generates factors for a given number.

  • The program takes an integer on the standard input and prints a string of integers. That string contains the prime factors in numerical sequence.
Example
  • 1 -> "" (empty string)
  • 3 -> "3"
  • 8 -> "2,2,2"
  • 9 -> "3,3"
  • 12 -> "3,4"
Bonus

How many lines can you reduce the main code down to? 10? 5? 3? You might find some clues in the answers to this SO Question
