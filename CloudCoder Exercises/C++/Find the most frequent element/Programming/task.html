Find the most frequent element

Find the most frequent element or elements in the list. The function will have two parameters:

first parameter: the list containing elements

second parameter: number of elements in the list

Example

find_most_frequent({1, 1, 2, 3}, 3) == set([1])

find_most_frequent({1, 1, 2, 2, 3}, 4) == set([1, 2])

Output

Make the function return a string containing the numbers. For example, if the elements are '1 and 2', then the output string must be "1 2 " with a space after each number.



Memory limit: 256 MB
Time limit: 5 seconds
