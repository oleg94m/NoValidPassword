Find all consecutive pairs that add to a given value

Complete the program so that it checks consecutive pairs of array elements to determine if their sum equals a specified value.

For example, for 5, 3, 4, 8, 5, 7, 5, 2, 5 and a desired value of 7 the output would be 

(1,2) (6,7) (7,8)

Another example, for 5, 3, 4, 8, 5, 7, 5, 2, 5 and a desired value of 12 the output would be 

(2,3) (4,5) (5,6)

Another example, for 5, 3, 4, 8, 5, 7, 5, 2, 5 and a desired value of 15 the output would be 

None found

Comments describe how to proceed. You should replace each "TODO" comment with appropriate code.

Hints:

  1. The array is named A[ ] and the number of elements it contains is stored in a variable len. The desired value is stored in a variable pairSum.
  2. Use a for loop to proceed through the array comparing each pair of array elements to the desired value.
  3. For found pairs, there is no space between the indicies inside the parentheses but a space between pairs of indicies.

Note that the test inputs include the number of values the array will contain as the first input and the desired value as the last input.



Memory limit: 256 MB
Time limit: 5 seconds
