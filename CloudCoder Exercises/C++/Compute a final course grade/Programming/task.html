Compute a final course grade

Complete the program so that it computes the exam average of three exams and quiz average of four quizzes, then computes the final course grade based on the following criteria:

  • If the exam average is higher than the quiz average then the final grade is simply the exam average.
  • Otherwise the final grade is computed as 75% exam average and 25% quiz average.

For example, for exam values 82, 78, 85 and quiz values 92, 85, 88, 79 the output would be 

Exam avg=81.67, Quiz avg=86.00, Final grade=82.75.

Anothe example, for exam values 86, 91, 84 and quiz values 72, 84, 78, 87 the output would be 

Exam avg=87.00, Quiz avg=80.25, Final grade=87.00.

Comments describe how to proceed. You should replace each "TODO" comment with appropriate code.

Hints:

  2. The variables examavg, quizavg, and finalgrd of type double have also been declared.
  3. A printf statement is already provided to produce the output in the expected format.
