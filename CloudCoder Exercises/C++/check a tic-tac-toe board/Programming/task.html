check a tic-tac-toe board

The program has a 3x3 array of int values, called board, which represents a tic-tac-toe board. When the program is run, it will read 9 integer values representing the configuration of the tic-tac-toe board: a 0 indicates an empty space, a 1 represents an X, and a 2 represents an O. The input values are read row by row: the first set of three values are the first row, the second set of three values are the second row, etc.

For example, consider the following input:

0 0 2 2 1 1 0 0 1

This input represents the following board: 

 | |O
-+-+-
O|X|X
-+-+-
 | |X

Your task is the following: determine whether the X player has won the game by placing three Xs in a single row, column, or diagonal. The output of the program should be one of the following lines: 

X wins in row N
X wins in column N
X wins in a diagonal
X has not won

In these messages, N is a row or column number. 0 is the first row or column, 1 is the second row or column, and 2 is the third row or column.

Example inputs and expected outputs:

InputExpected output 0 0 2 2 1 1 0 0 1 X has not won 1 2 0 2 1 2 0 0 1 X wins in a diagonal

Hints:

  • You can assume that X will have at most a single winning row, column, or diagonal
  • It may be helpful to use a boolean variable to keep track of whether or not a winning row/column/diagonal has been found: this will help the program know when to print the X has not won message
  • You may want to use a loop to check for winning rows and columns


Memory limit: 256 MB
Time limit: 5 seconds
