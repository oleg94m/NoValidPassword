Given 'nseq', produce an appropriate sequence

FizzBuzzSeq

First, read the value of a positive number nseq on the stdin. Given nseq print a numeric sequence from 1 to nseq. However, replace certain values if any of the following conditions are met:

  • If the value is a multiple of 3: use the value Fizz instead
  • If the value is a multiple of 5: use the value Buzz instead
  • If the value is a multiple of 3 & 5: use the value FizzBuzz instead

Example

  • 3 -> 1 2 Fizz
  • 5 -> 1 2 Fizz 4 Buzz
  • 7 -> 1 2 Fizz 4 Buzz Fizz 7


Memory limit: 256 MB
Time limit: 5 seconds
