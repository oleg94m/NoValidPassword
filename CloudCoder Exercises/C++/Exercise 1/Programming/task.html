Exercise 1

Complete the ProductIsEven() function below. This method should calculate the product of the two input values and return true (i.e. 1) if the product is an even number, and false (i.e. 0) if the product is an odd number.

For example, the product of 10 and 20 is 200, and this is an even number, so the function call ProductIsEven(10, 20) should return 1 (true).
