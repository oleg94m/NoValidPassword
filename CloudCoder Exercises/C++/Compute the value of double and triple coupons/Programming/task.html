Compute the value of double and triple coupons

Complete the program so that it outputs the redeemed value of a coupon based on the following criteria:

If the coupon's value is less than or equal to $0.75, then the redeemed value is tripled.

If the coupon's value is greater than $0.75 and less than or equal to $2.00, then the redeemed value is doubled.

If the coupon's value is greater than $2.00, then the redeemed value is face value.
