A Kata on the palindrome function

Is the string a palindrome?

Examine whether a given string is a palindrome or not.

check_if_palindrome ("abba") -> true

check_if_palindrome ("malayalam") -> true

check_if_palindrome ("bean") -> false

check_if_palindrome ("race car") -> true

check_if_palindrome ("I went there") -> false

The return value has to the bool basic data type that is supported in C++. Why does C++ need another data type? A great answer here!
