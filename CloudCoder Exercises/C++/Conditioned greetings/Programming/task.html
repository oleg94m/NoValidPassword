Conditioned greetings

Conditioned Greetings

  • If the name on the stdin is a girl ("girl Mary") then greet as "Hello Ms Mary!"
  • If the name on the stdin is a boy ("boy John") then greet as "Hello Mr John!"

Example

"girl Mary" -> "Hello Ms Mary!"

"boy John" -> "Hello Mr John!"

"boy Ramesh" -> "Hello Mr Ramesh!"

Clue

Use an index to gain access to the first letter of the gender word (the first word).

For example:

if type[30] = "male", then type[0] == 'm'
