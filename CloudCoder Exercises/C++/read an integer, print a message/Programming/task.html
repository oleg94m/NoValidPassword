read an integer, print a message

Complete the program so that it reads a single integer value using scanf, and then prints the message

You would like N tacos

where N is the integer value read.

For example, if the program reads the integer 3, then the output should be 

You would like 3 tacos

Comments describe how to proceed. You should replace each "TODO" comment with a single line of code.

Hints:

  1. Declare an int variable in which to store the integer value
  2. In your call to scanf, use the %i format specifier to read a single integer value
  3. Don't forget to put an ampersand (&) in front of the variable name in your call to scanf
  4. Use printf to print the message; as with scanf, use the %i format specifier to output the integer


Memory limit: 256 MB
Time limit: 5 seconds
