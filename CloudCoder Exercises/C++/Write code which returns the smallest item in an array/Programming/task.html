Write code which returns the smallest item in an array

For your program, write a function, named findMin, that returns an int and has 2 parameters: an array of ints and an int specifying the length of the array. Write your findMin function so that it returns the smallest item in the array that was passed in. You then need to write your main() function so that it creates two int arrays: one with a length of 3 and the other with a length of 5. Read in 3 values into the first array and use this in a call to findMin(). Print out the value findMin had returned and then a space. Then read in 5 numbers into your other array, use it in a call to findMin, and print out the value returned from that second call. YOU MUST PRINT OUT BOTH NUMBERS ON THE SAME LINE.



Memory limit: 256 MB
Time limit: 5 seconds
