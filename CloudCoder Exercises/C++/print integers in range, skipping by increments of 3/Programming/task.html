print integers in range, skipping by increments of 3

The program will receive two integer input values, start and end. The output of the program should be a line of text with all of the integer values between start and end, inclusive, such that each successive integer is 3 greater than the previous integer.

For example, if the input is 1 10, then the output should be 

1 4 7 10

Another example: if the input is 19 35, then the output should be 

19 22 25 28 31 34

Hints:

  • The program already declares variables for start and end, and uses scanf to read their values
  • Make sure to print a space after each integer
  • Think about how to generate only every third integer; planning the loop will be helpful


Memory limit: 256 MB
Time limit: 5 seconds
