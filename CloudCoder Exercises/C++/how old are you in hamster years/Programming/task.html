how old are you in hamster years?

Complete the program so that it reads an integer, which represents a person's age, and prints a message of the form 

Your age in hamster years is N

where N is the user's age in hamster years. One year is 25 hamster years, so if the program reads the integer 10, then the output should be 

Your age in hamster years is 250

Comments describe an approach to completing the program. Each TODO comment should be replaced with one line of code.

Hints:

  1. Declare two int variables: one for the user's age, and one for the user's age in hamster years.
  2. Use scanf to read the input into a variable using the %i format specifier. Don't forget to put an ampersand (&) before the variable name.
  3. Use an assignment to store the user's age converted to hamster years in a variable. The * operator performs multiplication.
  4. Use printf to print the message, using the %i format specifier to print the value in the variable containing the user's age in hamster years.
