How many jelly beans do you get?

Write a program that reads two integer values -- the first value will represent some number of jelly beans, the second value will represent some number of people. Imagine dividing up the jelly beans evenly amongst all of the people. Your program should compute the number of jelly beans that each person receives as well as the number of jelly beans that are leftover and print the results.

For example, if the input specifies 9 jelly beans and 4 people, your program should print the following message: 

Each person gets 2 jelly beans, with 1 leftover.

Hints:

  • Use scanf to read in the two integers
  • Note that jelly beans must be distributed as whole jelly beans. That is, a person cannot get two-thirds of a jelly bean. Recall that dividing two integer values will result in an integer result.
  • Use the modulus operator to compute the remaining number of jelly beans.


Memory limit: 256 MB
Time limit: 5 seconds
