check whether a value is in a specified range

Write just the definition of a function called inRange. It should take three parameters, all with type int:

  1. The first parameter is the minimum value in the range.
  2. The second parameter is the maximum value in the range.
  3. The third parameter is the value to be tested.

The function should return an int value: 1 if the value is in the range, 0 if the value is not in the range.

The range check should be inclusive: if the tested value is greater than or equal to the minimum, and less than or equal to the maximum, then it should be considered to be within the range.

Hints:

  • Make sure the name of the function is exactly as shown above, including capitalization
  • Declare a variable whose type is the return type (int). The last statement in the body of the function should return the value of this variable.
