Find all primes within a specified range

Write the function findPrimes() to print all prime numbers within a user specified range.

For example, for 1, 10 the output would be 

2 3 5 7

Another example, for 15, 37 the output would be 

17 19 23 29 31 37

Another example, for 24, 28 the output would be 

No primes found

Comments describe how to proceed. You should replace each "TODO" comment with appropriate code.

  1. The function findPrimes() should have 2 integer parameters but return no value. Do not forget to add an appropriate function prototype before int main() for the function.
  2. If the range contains 2, then 2 will be the first (and only) even prime number in the output. In this case, the lower range bound should then be set to 3.
  3. Any number n greater than 2 is not prime, if it is evenly divisible by any number between 2 and the sqrt(n) inclusive.
  4. A number n is evenly divisible by another number k (where k < n) if n == (n/k)*k since if it is not evenly divisible the integer division will cause truncation.
  5. The math library function for square root is sqrt().

Note: For a challenge, make the function more efficient by only checking odd values within the range (and greater than 2) for divisibility with odd values between 3 and sqrt(n) (consider how to handle the case when 3 is within the range). Furthermore, once an evenly divisible factor is found, the number is not prime, and thus no further factors need to be checked.
