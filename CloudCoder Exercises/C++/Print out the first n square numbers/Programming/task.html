Print out the first n square numbers

The square numbers can be found by computing the sum of the first n odd numbers (e.g., 1 = 1, 1 + 3 = 4, 1 + 3 + 5 = 9, etc.). I always wonder if this really works, but lack the math ability to prove it.

Write a program which reads in a number (this will be a whole number between 1 and 50, inclusive). Your program should then use a for loop to compute and print the sums of the first x odd numbers (where x is the number your program read in). After each number, you should print a space. To see how this works, consider the following output from my program having read in 5:

1 4 9 16 25


Memory limit: 256 MB
Time limit: 5 seconds
