Write a program which helps track POWs

According to the formal rules of war, combatants caught on the field of battle are only required to give their name, rank, date of birth, and serial number. Your program should declare a struct which declares 3 fields: a cString (with a maximum length of 80) named name, a cString (with a maximum length of 40 characters) named rank, and a long named serialNumber. Your program should begin by declaring an array of your struct type with a length of 5. Your program should then read in data for each of the 5 POWs (e.g., read the 0th entries data, then the 1st entry, and so on). Your program should then read in a name. For each POW in the array with that name, your program should print out their rank, a space, their serial number and another space (do NOT print out an endl).



Memory limit: 256 MB
Time limit: 5 seconds
