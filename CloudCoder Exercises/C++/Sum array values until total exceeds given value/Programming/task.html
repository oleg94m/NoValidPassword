Sum array values until total exceeds given value

Complete the program so that the sum of the first n values in an array such that the total does not exceed a specified value.

For example, for 5, 3, 4, 8, 5, 7, 5, 2, 5 and a desired sum of 24 the output would be 

4 values: 5+3+4+8 = 20

(Note that adding the next value would make the sum 25.)

Another example, for 5, 3, 4, 8, 5, 7, 5, 2, 5 and a desired sum of 60 the output would be 

9 values: 5+3+4+8+5+7+5+2+5 = 44

Another example, for 5, 3, 4, 8, 5, 7, 5, 2, 5 and a desired sum of 4 the output would be 

0 values: = 0

Comments describe how to proceed. You should replace each "TODO" comment with appropriate code.

Hints:

  1. The array is named A[ ] and the number of elements it contains is stored in a variable len. The desired sum is stored in a variable desSum.
  2. Use the variable total to track the running total of array elements.
  3. Use a while loop to proceed through the array comparing the current total to the desired sum or until there are no more array elements.
  4. Consider what to do with the last array value added once the total exceeds the desired sum.

Note that the test inputs include the number of values the array will contain as the first input and the desired sum as the last input.



Memory limit: 256 MB
Time limit: 5 seconds
