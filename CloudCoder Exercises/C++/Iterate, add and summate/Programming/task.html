Iterate, Add and Sum

  • Write a function add_one(int array[], int size) so that it takes an integer array as one of its parameters and its size as the other.
  • Iterate through the integer array (using a pointer), to increment each value in the array by value of 1.
  • Add the incremented value to a variable, say sum
  • Return the sum value

Example:

{1, 2, 3} -> 9

{2, 2} -> 6



Memory limit: 256 MB
Time limit: 5 seconds
