Swap elements of an array

Complete the program so that it swaps the first element of an array with the last, the second element with the next to last, etc. until the array is reversed.

For example, for 5, 3, 2, 6, 8 the output would be 

8 6 2 3 5

Another example, for 7, 9, 14, 3, 6, 2, 13, 1 the output would be 

1 13 2 6 3 14 9 7

Comments describe how to proceed. You should replace each "TODO" comment with appropriate code.

Hints:

  1. The array is named A[ ] and the number of elements it contains is stored in a variable len.
  2. Use a for loop to proceed through the array swapping pairs of values.
  3. For element A[i], consider what the index of the element it should be swapped with is.
  4. To swap two values A[i] and A[j], use a temp variable 
    temp = A[i]
    A[i] = A[j]
    A[j] = temp
  5. Consider when the loop should stop, i.e. when all the values have been swapped once.

Note that the test inputs include the number of values the array will contain as the first input.



Memory limit: 256 MB
Time limit: 5 seconds
