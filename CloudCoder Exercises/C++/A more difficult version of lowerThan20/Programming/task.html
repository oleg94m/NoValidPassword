A more difficult version of lowerThan20

Return false if the given non-negative number is further than n% of a multiple of m. So for example, for m = 20, n = 10, 78 and 79 return true, but 80 returns false.

lowerThanNofM(18, 20, 10) → true lowerThanNofM(24, 20, 20) → false lowerThanNofM(60, 30, 10) → false
