Compute and print out the AND truth table

Write a program which declares three variables of type bool. Set two of the variables to the value false, then assign the third variable the value of the using the AND operation on the other two variables (have the computer compute the AND operation; do NOT just assign the 3rd variable the value false). Print out the value of the 3rd variable and then a space. Repeat this process for the other 3 possible combinations of boolean values (e.g., false and true, true and false, true and true), where each time you print out the result of that AND operation and then a space. All of your output should be on a single line; do not print out a newline at the end.



Memory limit: 256 MB
Time limit: 5 seconds
