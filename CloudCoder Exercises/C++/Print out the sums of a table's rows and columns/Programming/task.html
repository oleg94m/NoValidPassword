Print out the sums of a table's rows and columns

For this problem, your program only needs a main() function. Your main() function should declare a 3-by-3 array of ints. Your program should read the value for the array entry at row 0 and column 0, then the value for the entry at row 0 and column 1, then the entry at row 0 and column 2. Your program should then read in the 3 values for row 1 and finally the 3 values for row 2. In your main, you should have the program compute the sum of the 3 values in row 0, then print a space. Next print the sum of the values in row 1 and a space. Then print the sum of the values in row 2 and a space. Then do the same thing for 3 columns (e.g., print the sum of the 3 values in columns 0, 1, and 2, with a space after each). MAKE SURE THAT YOU PRINT OUT ALL 6 NUMBERS ON THE SAME LINE WITH A SINGLE SPACE AFTER EACH.
