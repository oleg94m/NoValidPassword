Find the number of occurrences of a value in an array

Complete the program so that it determines the number of occurrences of a particular value in an array.

For example, for 5, 3, 2, 5, 8, 7, 5, 2, 4 and a desired value of 5 the output would be 

5 occurred 3 times

Another example, for 5, 3, 2, 5, 8, 7, 5, 2, 4 and a desired value of 8 the output would be 

8 occurred 1 times

Another example, for 5, 3, 2, 5, 8, 7, 5, 2, 4 and a desired value of 6 the output would be 

6 occurred 0 times

Comments describe how to proceed. You should replace each "TODO" comment with appropriate code.

Hints:

  1. The array is named A[ ] and the number of elements it contains is stored in a variable len. The desired value is stored in a variable desired.
  2. Use a for loop to proceed through the array comparing each array element to the desired value.
  3. Use the variable count to keep track of the number of occurrences. Be sure to initialize it properly.

Note that the test inputs include the number of values the array will contain as the first input and the desired value as the last input.



Memory limit: 256 MB
Time limit: 5 seconds
