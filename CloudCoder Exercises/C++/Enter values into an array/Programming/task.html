Enter values into an array

Complete the program so that it allocates an array named values[ ], repeatedly prompts the user to enter values which are stored into the array, until either:

  • A negative value is entered
  • More than 5 values are entered

and then displays the number of entered values along with the values entered.

For example, for 5, 3, 2, 6, -1 the output would be 

4 values entered: 5 3 2 6

Another example, for 7, 9, 14, 3, 6 the output would be 

5 values entered: 7 9 14 3 6

Comments describe how to proceed. You should replace each "TODO" comment with appropriate code.

Hints:

  1. Recall that the number given in an array declaration is the number of elements the array has.
  2. Use a separate variable for the user input and only store it into the array if the value is valid.
  3. Consider using a while loop that checks both conditions.
  4. Be sure not to count the negative value.


Memory limit: 256 MB
Time limit: 5 seconds
