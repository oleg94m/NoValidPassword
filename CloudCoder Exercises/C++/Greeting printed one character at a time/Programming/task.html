Greeting printed one character at a time

Functions

  • Read in a greeting. Read in a name to greet. Both on the standard in.
  • Print the greeting using both the strings (greeting and name).
  • Then call a function to print the two strings, one letter at a time, with a space in between.

Use strlen() (from the string library) to determine the length of the strings.

Example

  • "Hello" and "World!" -> "Hello World! H e l l o W o r l d ! "
  • "Hello" and "Dhoni!" -> "Hello Dhoni! H e l l o D h o n i ! "

Time limit: 5 seconds
