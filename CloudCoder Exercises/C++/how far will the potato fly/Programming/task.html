how far will the potato fly?

The PotatoFlinger3000™ will launch a standard potato with user-configurable velocity and angle. As a member of the engineering team, you have been asked to write a simulator that will determine how far the potato will fly from its launch site.

The distance d a projectile will fly can be computed with the formula

d = ( v2 sin 2θ) / g

where v is the velocity, θ is the angle from horizontal, and g is the downward acceleration due to gravity. (You can assume g = 9.81 m/s2.)

Your program should read in two data values:

  1. The potato velocity (m/s)
  2. The angle relative to horizontal (radians)

It should calculate the distance the potato will fly and print a message of the form 

The potato will fly N meters

where N is the distance. For example, if the input values are 12.0 and 1.047, then the output of the program should be similar to 

The potato will fly 12.715198 meters

Hints:

  • You'll need to include the stdio.h and math.h header files at the top of the program
  • Use scanf to read the input values
  • Use printf to print the output
  • The %lf format specifier is used for double values
  • The call sin(theta) will compute the sine of the angle theta (specified in radians)
  • You can ignore air resistance, the curvature of the earth, the size of the PotatoFlinger3000™, etc.


Memory limit: 256 MB
Time limit: 5 seconds
