print groups of 1s, 2s, etc...

The program takes two integer input values, numgroups and groupsize. Based on these values, the program should print a single line of text with numgroups groups of integer values, where each group has groupsize integers in it. The first group of integers should be 1s, the second group should be 2s, etc.

For example, if the input values are 4 2, then the output should be 

11223344

Another example: if the input values are 7 5, then the output should be 

11111222223333344444555556666677777

Hints:

  • Declare variables for the input values
  • Use scanf to read the input values
  • You will need nested loops. The outer loop is responsible for generating all of the groups. The inner loop is responsible for generating a single group.
  • Don't print any spaces between the integers.


Memory limit: 256 MB
Time limit: 5 seconds
