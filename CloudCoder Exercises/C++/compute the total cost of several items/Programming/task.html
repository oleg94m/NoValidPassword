compute the total cost of several items

Complete the program so that it reads several values using scanf which include

  • an integer for the number of hamburgers
  • a float for the cost of a hamburger
  • a float for the cost of an order of fries
  • a float for the cost of a soda
The program should then compute the total cost of the meal assuming the meal will include only one order of fries and one soda.

For example, (user input in bold) would produce the output 

How many hamburgers would you like? 3
How much does a hamburger cost? 2.50
How much does an order of fries cost? 1.00
How much does a soda cost? 1.25
The cost of lunch is $9.75

Comments describe how to proceed. You should replace each "TODO" comment with appropriate code.

Hints:

  1. Variables of appropriate types have already been declared
  2. For scanf, use the %i format specifier to read a single integer value
  3. For scanf, use the %f format specifier to read a single float value
  4. Don't forget to put an ampersand (&) in front of the variable name in your calls to scanf
  5. Create a new float variable to store the result of the computation using a meaningful name
  6. Use printf to print the message; as with scanf, use the %f format specifier to output the float answer
