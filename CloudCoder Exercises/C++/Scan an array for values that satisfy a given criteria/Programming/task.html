Scan an array for values that satisfy a given criteria

Write a program that scans the values of an integer array and prints out any value that falls in the range between 10 (inclusive) and 20 (inclusive). Put a space between each of the output values. The integer array has already been defined and initialized for you.

Since the array is predefined and there is no other user input, the output of this program should always be the same: 

17 10 13 20

Hints:

  • This program takes no input and therefore should always produce the same output
  • The integer array has already been defined for you.
  • Use a for-loop to traverse the entire array
  • Inside the for-loop, check to see if the value falls within the required range
  • Use the ARR_SIZE constant where appropriate


Memory limit: 256 MB
Time limit: 5 seconds
