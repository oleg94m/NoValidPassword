Write and use a simple function named average

Write a program that will use a user-defined function named average. Your function should not have any parameters and does not return a value. For its body, average() should read in 5 decimal numbers and print out their average followed by a space formatted to show exactly 2 values to the right of the decimal point. Your program MUST print everything out on 1 line. Write your program's main() function to use a loop which repeats 5 times. The loop's body should simply call the average() function you wrote.



Memory limit: 256 MB
Time limit: 5 seconds
