find the length of the hypotenuse

In a right triangle, the lengths of the sides of the triangle satisfy the equation

A2 + B2 = C2

where C is the hypotenuse of the triangle. This means that C is equal to the square root of the sum of the squares of A and B.

Complete the program so that it reads the lengths of the two non-hypotenuse sides of the right triangle. The program should then print the length of the hypotenuse. For example, if the input is the values 5.0 and 12.0, then the output of the program should be 

The hypotenuse is 13.000000

Hints:

  • You will need to include the stdio.h and math.h header files at the top of the program
  • The variables for the lengths of the sides of the triangle should allow fractions (not just integers)
  • Use scanf to read the input values
  • Use printf to print the output
  • The call sqrt(n) computes the square root of n
  • The expression n * n computes the square of n


Memory limit: 256 MB
Time limit: 5 seconds
