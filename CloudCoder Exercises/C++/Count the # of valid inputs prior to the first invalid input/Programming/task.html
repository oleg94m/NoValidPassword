Count the # of valid inputs prior to the first invalid input

Write a program that continually reads input values from a sequence of numbers until a 'termination' value is detected. Once the 'termination' value is detected, your program should print out the number of values that were seen prior to the 'termination' value. The total number of values in the numeric sequence is unknown and the 'termination' value can appear anywhere in the sequence.

The value -1 is the 'termination' value. All other integer values should be considered valid input and counted as such. When the value -1 is detected in the input sequence, your program should immediately stop reading values and print the number of valid values that were read.

For example, if the input sequence to your program is the following: 

10 -1
then your program should print: 
1 Valid

Another example, if the input sequence to your program is the following: 

10 19 8 -1 77 18 1001
then your program should print: 
3 Valid

Another example, if the input sequence to your program is the following: 

-1
then your program should print: 
0 Valid

Hints:

  • Since the number of values in the input sequence is unknown, you should use a while-loop, NOT a for-loop.
  • Think about when the while-loop should terminate and use that as the condition.
  • You can assume that all test inputs will include a termination value somewhere in the sequence. That is, you do NOT have to write your program to handle the case where the input sequence does not contain the value -1.
  • The termination value can appear anywhere in the numeric sequence. It may be the first value, the last value, or somewhere in the middle of the sequence.


Memory limit: 256 MB
Time limit: 5 seconds
