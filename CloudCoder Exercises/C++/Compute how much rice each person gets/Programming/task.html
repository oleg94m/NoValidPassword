Compute how much rice each person gets

Write a program that reads two values -- the first value is an integer and will represent how much rice is available to divvy up (in ounces), the second value is an integer and will represent some number of people amongst which the rice should be divided. After reading the two values your program should compute how many ounces of rice each person receives if the rice if divvied up evenly and print out the result to a precision of two decimal places.

For example, if the input specifies 17 ounces of rice and 3 people, your program should print the following message: 

Each person receives 5.67 ounces of rice.

Hints:

  • When computing the result, you will need to cast your integer inputs to either floats or doubles
  • The result of your computation must be stored in either a float or a double
  • Use the output precision specifier (from lecture 2) to print the appropriate number of decimal places


Memory limit: 256 MB
Time limit: 5 seconds
