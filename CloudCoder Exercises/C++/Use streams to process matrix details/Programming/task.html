Matrix Sum

Inspired by Stringstream Tutorial - C++ Tutorials | Dream.In.Code http://buff.ly/1ldwwpE

Input parameters include

matrix_in  - matrix representation in a string format
dimension  - either 'r' or 'c' indicating row or column
index      - i'th row or i'th column

Process matrix details in a string and return the sum of the numbers in a specific row or column.

For example: 

matrix_sum ("1 2 3\n4 5 6", 'r', 0) -> 6
matrix_sum ("1 2 3\n4 5 6", 'r', 1) -> 15
matrix_sum ("1 2 3\n4 5 6", 'c', 0) -> 5
