print requested number of specified integers, repeat

Put on your programming hat: this exercise is challenging!

The program will read a series of integer input values of the form

which howmany which howmany ... -1

Each which value specifies a particular integer that should be printed. Each howmany value specifies how many of that integer should be printed. If a which value is -1, then that means that the input has ended and the program should stop.

For example, if the input to the program is 2 4 3 7 -1 then the output should be 

22223333333

Another example: if the input to the program is 8 4 5 1 9 2 -1, then the output should be 

8888599

Hints:

  • Use an outer while loop, where each iteration should either read which and howmany values, or read a -1
  • If a -1 is read, set a variable that will cause the loop to terminate
  • If which and howmany values are read, use a for loop to print the requested number of integer values
  • Do not assume that if a -1 is read that it will be followed by another integer value. So, unconditionally reading two integer values on each iteration of the while loop won't work
  • Make sure all of the output is printed on a single line
  • Don't print spaces between the output values


Memory limit: 256 MB
Time limit: 5 seconds
