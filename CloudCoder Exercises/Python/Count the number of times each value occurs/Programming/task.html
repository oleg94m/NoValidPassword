Write a function that takes a list of ints as a parameter and returns a dictionary containing the counts of the number of times each value occurs (keys are the number and the values are the counts) For example: 
countNums([1,2,3,1,2,4]) => {1:2, 2:2, 3:1, 4:1}
    countNums([1,2,3,1,1,2,1]) => {1:4, 2:2, 3:1}
    countNums([78]) => {78:1}

Sample Input:
[1,2,3,1,2,4]

Sample Output:
{1:2, 2:2, 3:1, 4:1}



Memory limit: 256 MB
Time limit: 5 seconds
