Write a function that takes a list of strings named items and a dictionary named d with strings as both the keys and the values. Return a new list containing all of the strings from items , but if any string in items occurs as the key in d replace it with the value. For example: 
swapper(['ab','bc','dog'], {'ab':'xyz'}) => ['xyz', 'bc', 'dog']

Sample Input:
['ab','bc','dog'], {'ab':'xyz'}

Sample Output:
['xyz', 'bc', 'dog']



Memory limit: 256 MB
Time limit: 5 seconds
