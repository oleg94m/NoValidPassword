Write a function that takes a String that consists only of the capital letters A, C, G, and T, and returns the number of times each letter occurs. You can return multiple values by putting them in parentheses, like this: 
return (1, 2, 8, 4)
Be sure to return the counts in the order ACGT. For example: 
countBasePairs('GATTACA') => (3, 1, 1, 2)
countBasePairs('') => (0, 0, 0, 0)
countBasePairs('GGTCAGGCCCCT' => (1, 5, 4, 2)

Sample Input:
'GATTACA'

Sample Output:
(3, 1, 1, 2)



Memory limit: 256 MB
Time limit: 5 seconds
