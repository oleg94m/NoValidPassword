Write a function that takes a list of ints and returns the index (index, not the value) of the maximum element. If the max value occurs multiple times, return the index of the first occurrence. Do not use the max() function. The max of an empty list is undefined, so your function can crash if given an empty list. 
maxIndex([5,5,5,5])=>0
    maxIndex([5,15,10,20,25])=>4
    maxIndex([-3,-2,-1,-6,-9,-11])=>2

Sample Input:
[5,5,5,5]

Sample Output:
0
