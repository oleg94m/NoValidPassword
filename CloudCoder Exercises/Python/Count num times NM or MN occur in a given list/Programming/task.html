Write a function that takes a list of ints and two ints, N and M, and returns the number of times NM or MN occurs in order in the given list. For example: 
countNMorMN([1,2,3,4,5], 2, 1) => 1
    countNMorMN([1,2,3,4,5], 1, 2) => 1
    countNMorMN([1,2,1,2,1], 1, 2) => 4
    countNMorMN([-3, 8, 9, 12, 8, 12, 9], 12, 8) => 2

Sample Input:
[1,2,3,4,5], 2, 1

Sample Output:
1
