Write a function that takes a list of ints and returns True if the list is sorted from lowest to highest, and false otherwise. Assume that an empty list is sorted. For example: 
isSorted([1,2,3,4,99, 123, 2342]) => True
    isSorted([1,2,3,4,999, 50, 2342]) => False
    isSorted([]) => True

Sample Input:
[1,2,3,4,99, 123, 2342]

Sample Output:
True



Memory limit: 256 MB
Time limit: 5 seconds
