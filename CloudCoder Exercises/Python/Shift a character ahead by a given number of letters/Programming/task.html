Write a function that takes a single character and an int key as parameters, and then shifts the character ahead by the given number of spots, wrapping around from z to a. For simplicity, assume all input chars are lower-case letters. For example: 
shiftChar('a', 4) => 'e'
shiftChar('z', 1) => 'a'
shiftChar('z', 26) => 'z'

Sample Input:
'a', 4

Sample Output:
'e'



Memory limit: 256 MB
Time limit: 5 seconds
