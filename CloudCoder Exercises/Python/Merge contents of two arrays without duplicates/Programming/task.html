Merging sorted arrays

GTG 14.1
Given sorted arrays arr1 and arr2 of lengths n and m respectively, return an array out containing the union of the elements of both arrays, without any duplicates.

  • Use brute force to quickly establish the passing of test cases
  • Use binary search to search through arr1 (the bigger of the two) between inserting the element in arr2 into out

Example

[1, 1], [] -> [1]
[1], [2] -> [1, 2]
[1, 2], [2], -> [1, 2]
[3, 4], [1, 2] -> [1, 2, 3, 4]
[3, 4, 4], [1, 1, 2] -> [1, 2, 3, 4]
[1, 2, 3, 6, 7, 8], [4, 4] -> [1, 2, 3, 4, 6, 7, 8]


Sample Input:
[1], [2]

Sample Output:
[1, 2]



Memory limit: 256 MB
Time limit: 5 seconds
