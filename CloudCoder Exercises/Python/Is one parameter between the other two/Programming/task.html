Write a method called isBetween that takes three parameters and returns true if the first number is between the second two, inclusively, and false otherwise. 
isBetween(3, 3, 3) should return true
isBetween(4, 7, 9) should return false
isBetween(2, 1, 3) should return true
isBetween(2, 3, 1) should return true

Sample Input:
3, 3, 3

Sample Output:
True



Memory limit: 256 MB
Time limit: 5 seconds
