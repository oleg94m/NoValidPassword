Write a function that takes two lists of ints that are the same length, and returns the number of values in the second list that are greater than the value at the same index in the first list. For example: 
numGreaterAtIndex([1,2,3], [7, 8, 9]) => 3
    numGreaterAtIndex([7, 8, 9], [1, 2, 3]) => 0
    numGreaterAtIndex([7, 8, 9], [9, 0, 53]) => 2

Sample Input:
[1,2,3], [7, 8, 9]

Sample Output:
3



Memory limit: 256 MB
Time limit: 5 seconds
