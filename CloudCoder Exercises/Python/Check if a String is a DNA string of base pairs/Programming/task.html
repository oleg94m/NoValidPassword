Write a function that takes a string as a parameter, and returns True if the string is a DNA string (i.e. only contains the capital letters A, C, G, T), and False otherwise. A DNA string must be at least 1 character in length, so the empty string is not a DNA string. For example: 
isDNAString('GATTACA') => True
isDNAString('GATTACA 2: The Return of Ethan Hawke') => False
isDNAString('') => False

Sample Input:
'GATTACA'

Sample Output:
True



Memory limit: 256 MB
Time limit: 5 seconds
