Add a Closed Interval

Write a function which takes as input a list myl of disjoint closed intervals with integer endpoints, sorted by increasing order of left endpoint, and an interval interval, and returns the union of interval with the intervals in A, expressed as a union of disjoint intervals.

Example

[(1, 2)], (3, 4)-> [(1, 2), (3, 4)]
[(3, 4)], (1, 2)-> [(1, 2), (3, 4)]
[(0, 2), (3, 6), (7, 7), (9, 12)], (1, 8) -> [(0, 8), (9, 12)]


Sample Input:
[(1, 2)], (3, 4)

Sample Output:
[(1, 2), (3,4)]



Memory limit: 256 MB
Time limit: 5 seconds
