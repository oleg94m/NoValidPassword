Write a function that takes three parameters: a sequence of DNA called seq, a gene called g1, and a gene called g2. Return all of the strings that occur between an occurrence of g1 and the nearest occurrence of g2 that comes after it. If g1 occurs without g2, return the empty string. If there are extra occurrences of g2, use the first one. If there are extra occurrences of g1, use the last one. For example: 
betweenGenes('ATAATCGTGGCCA', 'AAT', 'GCCA') => 'CGTG'
betweenGenes('CGTGATCGCAATGGGGCAGTC', 'AT', 'CA') => 'CGGGGG'
betweenGenes('ATAATCGTGGCCA', 'AAT', 'GCCA') => 'CGTG'

Sample Input:
'ATAATCGTGGCCA', 'AAT', 'GCCA'

Sample Output:
'CGTG'



Memory limit: 256 MB
Time limit: 5 seconds
