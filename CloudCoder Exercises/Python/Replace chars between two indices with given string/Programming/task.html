Write a function that takes four parameters (a string s1, two ints, i and j, and another string s1) and replaces all of the characters of s1 between i (inclusive) and j (exclusive) with s2. If j is equal to or longer than the length of s1, replace everything in s1 from i to the end of the string with s2. For example: 
replaceAtIndex('knox college', 4, 10, ' villa') => 'knox village'
replaceAtIndex('knox college', 5, 100, 'rocks') => 'knox rocks'
replaceAtIndex('monmouth college', 0, 100, '') => ''

Sample Input:
'knox college', 4, 10, ' villa'

Sample Output:
'knox village'



Memory limit: 256 MB
Time limit: 5 seconds
