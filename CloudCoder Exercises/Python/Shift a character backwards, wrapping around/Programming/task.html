Write a function that takes a single character and an int key as parameters, and then shifts the character backwards by the given number of letters, wrapping around from a back around z. For simplicity, assume all input chars are lower-case letters.
Note that this it is harder to shift backwards than forwards. You may need to add extra multiple of 26 so that you don't get back a negative value when subtracting off the key.
For example: 
unshiftChar('a', 4) => 'w'
unshiftChar('z', 1) => 'y'
unshiftChar('z', 26) => 'z'

Sample Input:
'a', 4

Sample Output:
'w'



Memory limit: 256 MB
Time limit: 5 seconds
