Write a method that takes an int as a parameter and returns true if the parameter is both positive (i.e. greater than 0) and also strictly less than 10. For example: 
positiveLessThan10(10) => False
positiveLessThan10(-5) => False
positiveLessThan10(5) => True

Sample Input:
10

Sample Output:
False



Memory limit: 256 MB
Time limit: 5 seconds
