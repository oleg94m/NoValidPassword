Given an array of Strings, return true if the array of Strings is in alphabetical order.

For example: 
isAlphabetical( {"a", "b", "c"} ) => true

Note that you can compare chars with < to see if one comes before the other one (i.e. by comparing ASCII values).

Also, shorter strings come before longer strings whenever the shorter string is a subset of the longer one. So, "a" comes before "abc".

Finally, Strings which are identical are in alphabetical order.


Sample Input:
a b c

Sample Output:
true



Memory limit: 256 MB
Time limit: 8 seconds
