Write a function (statis method) that takes an array of ints as a parameter, and returns true if there are no duplicate ints in the array (i.e. no int value appears twice), and false otherwise.

For example: 
noDuplicates(new int[] {1,2,3} ) => true

Sample Input:
1 2 3

Sample Output:
true



Memory limit: 256 MB
Time limit: 8 seconds
