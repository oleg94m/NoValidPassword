Write a function (static method) called sumAbove that takes an array of ints and an int as parameters and returns the sum of the values in the array that are strictly greater than the given number.

For example: 
sumAbove({5, 8, 11, 2, 10}, 8) => 21

Sample Input:
5 8 11 2 10
8

Sample Output:
21



Memory limit: 256 MB
Time limit: 8 seconds
