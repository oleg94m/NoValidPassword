Complete the isPalindrome method so that it returns true if the string passed as the parameter is the same if its characters are reversed. For example, "abba" is a palindrome, but "abbc" is not a palindrome.

Important: your implementation must be recursive: do not use a loop. Note that CloudCoder will probably not check whether your method is recursive: you're on the honor system.

Think about what base case or cases would be appropriate. Make sure that your recursive call or calls make progress towards a base case.

The method should not ignore spaces, punctuation, or case differences. For example, "abbA" should not be considered a palindrome.

You can use the length() method to get the length of a string. The charAt(int) method gets the character at the given index in a string. The substring(int, int) method gets a substring from the given start index (inclusive) to the given end index (exclusive). A correct solution will probably use all of these methods.

"abba"

Sample Output:
true
