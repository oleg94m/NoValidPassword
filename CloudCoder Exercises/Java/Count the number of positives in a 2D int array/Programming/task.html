Write a function (static method) that counts the number of positive values in a 2D int array (i.e. int[][]). The 2D int array may not be square or rectangular, but you can assume that none of the rows is null. For exampe: 
superSumPositives( new int[][] { {-1,0,1}, {-2,-1}, {0}}) => 1
superSumPositives( new int[][] { {-1,0,1}, {1,2,3}, {0,0,0}}) => 4
superSumPositives( new int[][] { {-1,0,-1}, {-2,-1}, {-1,-2,-3}}) => 0

Sample Input:
new int[][] { {-1,0,1}, {-2,-1}, {0}}

Sample Output:
1
