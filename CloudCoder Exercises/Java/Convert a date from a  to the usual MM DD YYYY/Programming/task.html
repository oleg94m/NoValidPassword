Write a program that takes a date string formatted as YYYY-MM-DD as input and converts and outputs it as MM/DD/YYYY. For instance, the input 2007-07-21 will result in the following output 07/21/2007. Print Incorrect input if the date is not possible. Assume that every month has 31 days and that the year, month or day cannot be 0 (or 00).
Sample Input:
"2012-09-28"

Sample Output:
"09/28/2012"



Memory limit: 256 MB
Time limit: 8 seconds
