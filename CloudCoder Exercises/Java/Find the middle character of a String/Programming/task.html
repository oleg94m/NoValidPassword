Find the middle character of a String. The middle character of an empty String is also the empty String. Otherwise, the middle char of Strings of an odd length is the middle char, while the middle char of Strings of an even length is actually the middle two chars.

For example: 
middleChar("asdf") => "sd"

Sample Input:
a

Sample Output:
a



Memory limit: 256 MB
Time limit: 8 seconds
