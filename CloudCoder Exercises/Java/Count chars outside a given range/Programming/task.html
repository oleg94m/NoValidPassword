Count the number of times a given char occurs OUTSIDE a given range of a String parameter (i.e. starting at a given start index, and up to but not including and endindex).
For example: 
countCharsOutsideRange("abcdaba", 'a', 1, 4) => 3

Sample Input:
abcdaba a 1 4

Sample Output:
3
