Write a function (static method) that takes an array of ints as a parameter, and returns true if the array contains the same number of positive and negative values. Remember that zero is neither positive nor negative.

For example: 
sameNumPosNeg(new int[] {1,0,-1}) => true

Sample Input:
1 0 -1

Sample Output:
true



Memory limit: 256 MB
Time limit: 8 seconds
