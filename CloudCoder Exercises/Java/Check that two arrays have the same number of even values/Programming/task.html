Write a function (static method) that takes two int arrays as parameters, and returns true if they both have the same number of even values, and false otherwise.

For example: 
sameNumEvens( {1,2,3}, {4,5,6} ) => false

Sample Input:
1 2 3
4 5 6

Sample Output:
false



Memory limit: 256 MB
Time limit: 8 seconds
