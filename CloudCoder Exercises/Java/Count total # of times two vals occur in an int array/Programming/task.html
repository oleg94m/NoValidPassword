Given an array of ints and two ints, count the total number of times each int occurs in the array. If the two int parameters are the same, you should only count them once.

For example: 
numTimes2({1,2,3,4,2,1}, 1, 2) => 4

Sample Input:
1 2 3 4 2 1
1 2

Sample Output:
4



Memory limit: 256 MB
Time limit: 8 seconds
