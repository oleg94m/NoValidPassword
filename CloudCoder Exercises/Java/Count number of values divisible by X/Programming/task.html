Write a function (static method) that takes an array of ints and an int named x as parameters, and returns the number of values in the array that are divisible by x.

For example: 
countDivisibleByX( {1,2,3,4,5,6}, 2) => 3

Sample Input:
1 2 3 4 5 6
2

Sample Output:
3



Memory limit: 256 MB
Time limit: 8 seconds
