Given an array of ints, return the number of "triples" in the array. A "triple" is three consecutive ints in increasing order differing by 1 (i.e. 3,4,5 is a triple, but 5,4,3 and 2,4,6 are not).

For example: 
numTriples({1,2,4,5,6,7}) => 2

Sample Input:
1 2 4 5 6 7

Sample Output:
2



Memory limit: 256 MB
Time limit: 8 seconds
