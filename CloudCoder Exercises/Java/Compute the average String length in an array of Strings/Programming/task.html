Write a function (static method) that takes an array of Strings as a parameter and returns the average length of the Strings (as a double). If a String is null, assume its length is 0.

For example: 
averageStringLength({"hello", "world"}) => 5.0

Sample Input:
"hello" "world"

Sample Output:
5.0



Memory limit: 256 MB
Time limit: 8 seconds
