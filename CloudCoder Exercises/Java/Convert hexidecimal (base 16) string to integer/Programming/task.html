Use recursion to convert a string of hexidecimal digits to an integer. Don't use a loop!

The "digits" of a hexidecimal number are in base 16. 0-9 are interpreted as usual, and the letters "a" through "f" represent the values 10 through 15. So, for example,

c2b

Represents the integer 3115, because "c" = 12, "b" = 11, and 12*162 + 2*161 + 11*160 = 3115.

You may assume that all of the letters in the input string will be lower case.

Hints:

  • Think about what would be an appropriate base case. The first thing the method should do is check a base case.
  • The length method gets the length (number of characters) of a string.
  • Use the charAt method to get the character value at a particular index in the string.
  • The substring method returns a string that contains all of the characters from a start index (inclusive) to and end index (exclusive). So, for example, s.substring(0, s.length()-1) would return a string with all of the characters except the last character.

Sample Input:
c2b

Sample Output:
3115
