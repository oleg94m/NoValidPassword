Write a static method called ascendingOrder that takes three ints (a, b, c) and a boolean (eq) as parameters. The method should return true if eq is true and a, b, c are in ascending order and possibly equal. The method should return false if a, b, c are not in ascending order. If eq is true, then a, b, and c are allowed to be equal for the purposes of determining if they are in ascedning order.

For example:

ascendingOrder(2, 2, 2, true) -> true


Sample Input:
2 2 2 true

Sample Output:
true



Memory limit: 256 MB
Time limit: 8 seconds
