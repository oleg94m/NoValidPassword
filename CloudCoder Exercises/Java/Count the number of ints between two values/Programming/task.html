Write a function (static method) that takes an array of ints and two ints a and b as parameters, and returns the number of values that are greater than or equal to a, and less than or equal to b. If a is greater than b, return 0.

For example: 
countNumBetween( {1,2,3,4,50,100}, 25, 75) => 1 (since 50 is the only number between 25 and 75)

Sample Input:
1 2 3 4 50 100
25 75

Sample Output:
1



Memory limit: 256 MB
Time limit: 8 seconds
